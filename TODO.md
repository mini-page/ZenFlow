# 📝 ZenFlow Development Roadmap

This file tracks the upcoming features, bug fixes, and architectural improvements for ZenFlow.

## 🟩 High Priority (Core Polish)
- [ ] **Mobile Swipe Gestures**: Implement edge-swipes to return to the dashboard from any view.
- [ ] **Soundscape Cross-fading**: Smoothly transition between different ambient sounds to avoid sudden audio cuts.
- [ ] **Data Export/Import**: Allow users to backup their "Garden Data" (tasks, sessions, custom sounds) as a JSON file.
- [ ] **Custom Garden Themes**: Add a "Theme Selector" to change the garden's color palette (e.g., Midnight, Autumn, Spring).

## 🟦 Medium Priority (Enhanced Features)
- [ ] **Collaborative Focus Rooms**: Create simple "Deep Work" rooms where users can see anonymous "Garden Pulsing" from other active users.
- [ ] **Advanced Analytics**: Interactive line charts for 30-day productivity trends.
- [ ] **Hydration Reminders**: Smart browser notifications if no water has been logged for 2+ hours.
- [ ] **Achievement System**: Unlockable garden elements (e.g., Rare Flowers, Statues) for long-term streaks.

## 🟨 Low Priority (Nice to Have)
- [ ] **Audio Visualizer**: A faint, elegant wave animation in the Sound Sanctuary reacting to active audio.
- [ ] **Weather-Sync SFX**: Automatically play gentle rain sounds if the local weather sync detects rain.
- [ ] **Keyboard Shortcut Guide**: A modal (press `?`) showing all available power-user shortcuts.
